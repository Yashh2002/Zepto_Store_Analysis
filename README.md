#  📊 Zepto Sales Analytics Dashboard (Power BI)

An interactive **Power BI dashboard** analyzing $900K+ sales data across 6,400+ products to **uncover insights related to product performance, outlet distribution, and revenue trends**.
The dashboard enables data-driven decision-making using KPIs, filters, and visual analytics.

## 🚀 Project Highlights
- Analyzed $900K+ sales dataset across 6400+ products
- Designed interactive Power BI dashboard
- Created KPIs using DAX & Power Query
- Built visualizations for product category performance
- Implemented filters & slicers for dynamic analysis
- Identified top-performing outlet tiers and product categories
- Enabled multi-dimensional business insights

## 🧠 Business Problem

### Retail businesses need to:
- Identify high-performing products
- Understand outlet performance
- Optimize inventory distribution
- Improve revenue generation strategies

### This dashboard helps stakeholders quickly analyze trends and take data-backed decisions.

## 🛠️ Tech Stack

| Category | Tool |
|---------|------------|
| Data Visualization | 	Power BI |
| Data Processing |	Power Query |
| KPI Calculations |	DAX |
| Dataset Format |	Excel (.xlsx / .csv) |

## 📊 Key Performance Indicators (KPIs)

| Total Sales        : $900K+ |
| Average Sales      : Calculated using DAX |
| Product Count      : 6400+ |
| Outlet Count       : Multiple locations analyzed |


## 📈 Dashboard Insights
### 1️⃣ Product Category Performance
Identifies highest revenue generating product categories
Highlights demand distribution
### 2️⃣ Outlet Tier Analysis
Compares Tier 1, Tier 2, Tier 3 outlet performance
Shows revenue contribution by outlet tier
### 3️⃣ Outlet Size Distribution
Analyzes Small / Medium / High size outlets
Identifies which outlet size drives most sales
### 4️⃣ Location-wise Sales
Sales performance across different outlet locations
### 5️⃣ Multi-Dimensional Filtering
Users can filter dashboard based on:
- Outlet Location
- Outlet Size
- Product Category
- Outlet Tier

## 🏗️ Dashboard Architecture

Sales Dataset → Data Cleaning (Power Query) → KPI Calculation (DAX) → Data Modeling → Interactive Dashboard → Business Insights

## 🔍 Key Insights Generated
- Identified top-selling product categories contributing maximum revenue
- Tier 1 outlets showed highest overall sales performance
- Medium size outlets contributed significant portion of revenue
- Certain product categories showed consistent high demand across locations
- Sales trends helped identify optimization opportunities
